02 — Workflow

Generate.
Iterate.
Import.

The addon is built around three phases. Each phase stays inside Blender, uses native data blocks, and never forces an external app.

  1. 01

    Brief

    Write a prompt or drop an image. Set scale, polycount target, and style references. Save as a preset.

    • Prompt or image input
    • Scene-aware scale
    • Reusable presets
  2. 02

    Iterate

    Generate variants in parallel. Compare side by side, lock parts you like, regenerate the rest. No file juggling.

    • Variant grid
    • Region locking
    • Seed control
  3. 03

    Import

    Bring the chosen result into your scene as a real mesh with materials, UVs, and a clean modifier stack.

    • Native mesh data
    • Editable shader graph
    • Non-destructive setup

06 — FAQ

Quick answers.

Does it work offline?

Generation requires an AI provider, so an internet connection is needed for that step. Everything else — managing assets, refining meshes, browsing the library — works offline.

Which provider does it use?

You bring your own key. The addon supports a small set of providers per modality. Keys are stored locally and never proxied through us.

Is the output usable in production?

Geometry is generated as quads with topology hints. Materials use the standard Principled BSDF. Heavy results can be retopologized in one click.

Can I use generated assets commercially?

That depends on the provider you use. The addon does not add restrictions on top. Check your provider's terms.

Will it ever stop being a Blender-only tool?

No. The addon is designed for Blender, and we have no plans to dilute that focus.
